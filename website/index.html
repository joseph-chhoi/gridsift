What is GridSift

Built for medical imaging researchers

When hospitals export imaging data, researchers receive hundreds of mixed, unlabeled DICOM files with inconsistent naming conventions that vary by institution, scanner vendor, and country. Before any actual research can begin, someone has to manually sort through all of it — which takes days on large datasets.

GridSift solves this with a two-stage pipeline: a physics-based filter that uses vendor-agnostic MRI acquisition parameters (TE, TR, flip angle, inversion time) to classify unambiguous series, and a local LLM that handles the rest. Uncertain series are explicitly flagged for human review — never silently mislabeled.

🔒

Fully local

No data leaves your machine. No API keys, no internet connection required, no cost per use. Safe for sensitive clinical data.

⚡

Physics-first

Uses vendor-agnostic numeric MRI parameters as primary signal — works regardless of how the hospital named the series.

🚩

Uncertainty flagging

Uncertain series are flagged for human review, never silently misclassified. Every decision is logged and auditable.

📁

Sorted output

Files are copied into labeled subfolders — T1, T2, FLAIR, DWI, DCE, flagged_for_review — ready for your pipeline.

📊

CSV export

Every classification is logged with confidence score and decision path. Export to CSV for full reproducibility.

🌐

Multi-site ready

Works across institutions, scanner vendors, and naming conventions. No retraining or configuration needed.

How to use GridSift

From download to sorted output in five steps.

1

Download GridSift and the model

Download the GridSift zip for your operating system from the Downloads page. Also download the Llama 3.2 3B model file (2GB). Extract the GridSift zip to a folder on your computer.

2

Place the model file inside the GridSift folder

Move the downloaded model file into the same folder as the GridSift executable. The app looks for the model file next to itself on startup.

📁 GridSift/
  GridSift.exe
  Llama-3.2-3B-Instruct-Q4_K_M.gguf ← place here
  ... other files
3

Open GridSift and select your folders

Launch GridSift.exe. Click Select Folder to choose your DICOM dump folder. Click Select Output to choose where sorted files will be saved. Optionally type a Study Name to create a named subfolder.

4

Run classification

Click Run Classification. GridSift scans all DICOM files, groups them by series, and classifies each one. A progress bar shows real-time status. Results appear in the table as each series is classified.

# GridSift processes your data:

Brain STIR → FLAIR (0.90) · physics
CBU_DTI_64D → T2 (0.85) · physics
RESTING_STATE → unknown (0.0) · flagged ⚠️
5

Review flagged series and export

Series highlighted in red were uncertain — review those manually. Click Export Results as CSV to save a full log. Your output folder now contains sorted subfolders ready for your research pipeline.
